# User Stories

## Blog
- Como usuário leigo, quero escolher um template de blog para começar rápido.
- Como usuário, quero editar meu blog sem precisar programar.
- Como usuário, quero publicar meus conteúdos diretamente no blog.

## Estratégia
- Como usuário, quero responder perguntas simples sobre meu negócio para receber uma estratégia de conteúdo.
- Como usuário, quero ver palavras sugeridas além das que já imaginei.
- Como usuário, quero entender o motivo pelo qual uma palavra ou tema foi sugerido.

## Conteúdo
- Como usuário, quero que a plataforma gere artigos completos para mim.
- Como usuário, quero aprovar antes de publicar se eu desejar.
- Como usuário, quero editar um artigo antes ou depois da publicação.

## Tendências
- Como usuário, quero ver oportunidades de conteúdo em alta para aproveitar movimentos do mercado.
- Como usuário, quero que essas oportunidades façam sentido para meu nicho e minha localização.

## Analytics
- Como usuário, quero entender quais conteúdos e palavras estão funcionando melhor.
- Como usuário, quero enxergar se o conteúdo está gerando clique e oportunidade de negócio.
- Como usuário, quero entender a diferença entre performance SEO e GEO.
